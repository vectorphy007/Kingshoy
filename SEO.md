# Kingshot Alliance Companion — SEO / AEO Guide

This document defines search engine optimization (SEO) and answer engine optimization (AEO) strategies.

Goals:

- rank guides in search engines  
- capture Kingshot-related search traffic  
- improve discoverability  
- improve internal linking  
- structure content for AI/search assistants  

---

# SEO Priorities

Highest SEO value pages:

1. Hero Guides  
2. Event Guides  
3. Formation Guides  
4. FAQs  
5. Calculator Pages  
6. Glossary / Mechanics Pages  

These pages can bring organic traffic.

---

# Core SEO Routes

Examples:

```plaintext id="w7u2q6"
/guides/heroes/[hero]
/guides/events/[event]
/guides/formations/[formation]
/guides/faq/[slug]
/calculators/[tool]
````

# Meta Description Rules

Every page should have meta descriptions.

# URL Rules

Good:

Bad:


# FAQ Strategy

Create FAQ pages targeting questions.

Examples:

```text id="t8u4p6"
What is the best rally hero in Kingshot?
```

```text id="h9v3m1"
How does Bear Hunt work in Kingshot?
```

```text id="k7r2n5"
How do I increase Rally Cap in Kingshot?
```

These are good for SEO and AEO.

# Internal Linking Rules

Every guide should link to:

* related guides
* relevant events
* relevant calculators
* related heroes

Bear Hunt guide links to:

* Bear Hunt calculator
* hero guides
* formations guide
